### 控制数字精度

| 函数名         | 描述        | 例子                        | 例子2                 |
| ---           | ---        | ---                        | ---                  |
| CEIL()        | 向上取整     | CEIL(72.001) = 73          |                      |
| FLOOR()       | 向下取整     | FLOOR(72.999) = 72         |                      |
